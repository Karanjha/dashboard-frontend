Introduction

Welcome to Dashboard, which aims to be an umbrella for utility services for students at IIT Kanpur.

At present, we have three services, namely, a student search, a calendar for timetable and event tracking, and a service to act as a parser for the 'share auto/tempo' facebook group.

Dashboard is written as a series of microservices, which run in their own docker, and communicate via JSON RPC. This means that we can keep extending Dashboard by adding more services.
